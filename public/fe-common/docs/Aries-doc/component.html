图标

所有字体图标必须放在<i>标签上，图标类只能应用在不包含任何文本内容或子元素的标签上;可设置font-size属性改变图标的大小

增加 icon-sm、icon-md、icon-lg、icon-xl中任何一个类可改变图标大小

  1. <i class = "search" > </i>
  2. <i class = "search icon-sm" > </i>
  3. <i class = "search icon-md" > </i>
  4. <i class = "search icon-lg" > </i>
  5. <i class = "search icon-xl" > </i>

更多图标请点击查看所有字体图标

表格

基本实例

为任意 <table> 标签添加 .table 类可以为其赋予基本的样式 — 少量的内补（padding）和水平方向的分隔线。
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
  1. <table class = "table" > ... </table>

条纹状表格

通过 .table-striped 类可以给 <tbody> 之内的每一行增加斑马条纹样式。
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
  1. <table class = "table table-striped" > ... </table>

带边框的表格

添加 .table-bordered 类为表格和其中的每个单元格增加边框。
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
  1. <table class = "table table-bordered" > ... </table>

紧缩表格

通过添加 .table-condensed 类可以让表格更加紧凑，单元格中的内补（padding）均会减小。
# First Name Last Name Username
1 Mark Otto @mdo
2 Jacob Thornton @fat
3 Larry the Bird @twitter
  1. <table class = "table table-condensed" > ... </table>
